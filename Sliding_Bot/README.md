# 슬라이딩 봇 퍼즐

모바일·PC 웹에서 즐기는 4x4 이진 추리 게임. 매 판 6개의 칸이 채워진 정답 격자가 무작위로 생성됩니다.

## 목표
- 4x4 격자에서 정확히 6칸을 맞추기.

## 격자 조작
- 격자 칸을 탭/클릭하면 `빈칸 → 채움 → ? → 빈칸`으로 순환합니다.
- `?`는 메모용으로 정답 채움으로 계산되지 않습니다.
- 정답 제출은 정확히 6칸을 채운 뒤 가능하며, 일치 여부와 맞춘 개수를 알려줍니다.

## 질문 입력 (7회)
- 각 시도는 `(행, 열, 방향1, 방향2, 방향3)`을 입력합니다. 방향 키는 ↑(U), →(R), ↓(D), ←(L).
- 제약: 같은 방향 연속 불가, 바로 반대 방향 연속 불가.
- 제출 후 오른쪽 칸에 결과 좌표가 표시됩니다. 총 7번까지 시도 가능합니다.

## 이동 규칙
- 격자는 좌우/상하가 이어져 있습니다(래핑).
- 이동 시 현재 행/열이 모두 비어 있으면 결과는 `🌀`로 표시되어 무한 이동을 방지합니다.
- 이동은 진행 방향으로 빈칸을 따라가다가 처음 만나는 채워진 칸 바로 앞에서 멈춥니다.
- 시작 칸이 채워져 있으면 그 턴 동안 격자를 반전해서 계산합니다(채워진 칸↔빈칸).

## 주요 버튼
- `새 게임`: 정답과 격자를 초기화합니다.
- `정답 제출`: 내가 채운 6칸을 정답과 비교합니다.
- `?`(튜토리얼): 규칙을 팝업으로 다시 보여줍니다.

## 실행
- `index.html`을 브라우저로 열면 바로 플레이할 수 있습니다. 별도 빌드나 설치가 필요 없습니다.
